Scrapes the Amity LMS date range into the same leads SQLite DB the caller uses (source=portal_scraper), with pacing from .env (SCRAPER_MIN_DELAY_SEC / SCRAPER_MAX_DELAY_SEC). While a job is Running, this page polls quickly so Rows scraped updates live. To watch the browser highlight each row as it is read, set SCRAPER_HEADLESS=false; with a visible window, SCRAPER_HIGHLIGHT_ROWS=auto (default) turns on row-by-row extraction with an orange outline on the current row. If the grid never loads data, try SCRAPER_BLOCK_HEAVY_RESOURCES=false.
